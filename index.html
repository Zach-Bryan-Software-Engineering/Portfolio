Projects

Here’s a showcase of some of my favorite projects.

Generative Dungeon Script
This project is a procedural dungeon generator built from scratch. It automatically creates dungeon layouts using pre-defined tiles and logic to ensure halls and rooms connect seamlessly. The system tracks recent turns to avoid collisions and can detect obstacles, allowing it to make intelligent decisions about where to generate the next section. This project demonstrates procedural generation, algorithmic problem-solving, and game design principles.

Lobby Hosting UI & Inventory System
This project demonstrates a robust user interface for hosting and joining game lobbies, paired with an integrated inventory management system. Despite its simple appearance, the UI handles complex interactions such as item dragging with precise mouse tracking, animated feedback, and sound effects. All player actions are stored in a persistent data system to ensure inventory and lobby states are saved across sessions. The lobby system also synchronizes data between players and servers, enabling seamless multiplayer experiences.

Super Smash Bros Remade – Final Project for Computer Science Principles
This project was created in a single week during finals. The assignment was open-ended, allowing the use of AI tools like ChatGPT to help bring ideas to life. I challenged myself to build a fully playable game without using a traditional game engine. While much of the code was generated with AI assistance, AI alone could not create a project of this scale. I provided the ideas, solutions, and direction, effectively stitching everything together into a cohesive game. The project took approximately 35 hours to complete and was a valuable learning experience in both coding and problem-solving.
